# Paypercut Open Source

Welcome to the official open source organization of **Paypercut**.

This is where we publish open source components from the **Paypercut ecosystem** — practical tools and integrations built to help developers, merchants, and partners work with our platform and related systems more effectively.

## Scope

Repositories in this organization may include:

- CMS plugins
- Ecommerce platform integrations
- SDKs and client libraries
- Developer tooling
- Reference implementations
- Examples and supporting utilities
- Other reusable open source pieces from our ecosystem

We believe useful infrastructure, integrations, and tooling should be easy to adopt, easy to understand, and easy to build on.

## About Paypercut
